\# 朋友圈军师 · 豆包AI分析助手



一个基于豆包大模型的朋友圈截图分析工具，帮助用户通过分析目标人物的朋友圈，获取追求建议。



\## 功能特点



\- 📸 上传5张以上朋友圈截图

\- 🎯 豆包大模型多模态分析

\- 💡 简短有效的追求建议

\- 🎨 简洁美观的H5界面

\- 🔒 图片仅用于分析，即时清理



\## 技术栈



\- 前端：HTML5 + CSS3 + JavaScript (原生)

\- 后端：Flask + Python

\- AI模型：豆包 doubao-seed-1.6-thinking
